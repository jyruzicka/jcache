# JCache

JCache is very simple. Use `JCache::Cache` instances like hashes, and save/load them as you wish.

    c = JCache::Cache.new('foo') # or JCache::Cache['foo']
    c[:key] = 3
    c.save

    d = JCache::Cache.new('foo') # the same cache!
    d[:key] #=> 3
